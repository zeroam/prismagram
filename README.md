# Prismagram

Instagram clone with Express + Prisma + React and React Native

## User Stories

- [x] Create account
- [x] Request Secret
- [x] Confirm Secret (Login)
- [x] Like / Unlike a post
- [x] Comment on a post
- [x] Search by user
- [x] Search by location
- [x] Follow User
- [x] Unfollow User
- [x] Edit my profile
- [x] See user profile
- [x] See My profile
- [x] See the full post
- [x] Upload a post
- [x] Edit the post (Delete)
- [ ] Seed the feed
- [ ] See rooms
- [ ] See room
- [ ] Send private Message
- [ ] Receive Message (Realtime)
